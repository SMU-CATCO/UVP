# URVA Visualization Program (UVP)

## Program Overview
UVP is a user-friendly GUI designed for the visual mechanistic analysis of URVA output data, especially the curvature of the reaction path (RP) on the potential energy surface (PES) of the reaction complex (RC), the union of the reacting species. This interface enables users to interactively and simultaneously compare the change in the RP curvature with changes in the RC geometry and electronic energy as a function of the reaction coordinate. Cross-comparing the RP curvature with the RC properties, the user can better understand the chemical meaning of each URVA reaction phase, thereby enhancing the mechanistic analysis of each elementary step of a given reaction mechanism; for an overview of URVA and its previous applications, please see the following review article: https://doi.org/10.1039/D3CC01576A.

The GUI requires three input files: the RP curvature file (.dat format), the RC energy file (.csv format), and the RC geometry file (.xyz format). These output files are generated by our group's URVA2025 program and its associated post-processing scripts,<sup>1</sup> which will be provided upon request (contact email: ekraka@mail.smu.edu). Examples of these input files are provided in the `examples` subdirectory and are also discussed below.

## Installation and Usage

Clone the repository to your local computer and access the `UVP` directory
```Bash
git clone git@github.com:SMU-CATCO/UVP.git
cd UVP
```
Create the UVP conda environment from the uvp.yml environment file and then activate the environment
```Bash
conda env create -f uvp.yml
conda activate uvp
```
Copy the uvp.py file to a directory of your choice, such as your home directory
```Bash
cp uvp.py path/to/new/uvp/directory
```
In your local ~/.bashrc file, include an alias to run the uvp.py script
```Bash
alias uvp="python3 path/to/new/uvp/directory/uvp.py"
```
Afterwards, reload the ~/.bashrc file
```Bash
source ~/.bashrc
```
Now, you can use the `uvp` alias to automatically run the uvp.py script within any directory containing the three UVP input files. An example is given below.

## Example

Here, we will apply UVP to an elementary step of the ion-neutral, gas-phase reaction between acetonitrile (CH<sub>3</sub>CN) and the carbon monochloride ion (CCl<sup>+</sup>), which was previously examined with URVA.<sup>2</sup> More specifically, this elementary reaction is the unimolecular isomerization of the Van der Waals complex described by intermediate **5** to form intermediate **6** (see PES shown in Figure 1 of ref. [2]).

First, from the `UVP` directory, access the `example_1` subdirectory contained within the `examples` subdirectory
```Bash
cd examples/example_1
```
This directory contains the RP curvature file (originalkappa.dat), RC energy file (energy.csv), and RC geometry file (reaction.xyz) for the elementary reaction. These are the three input files need to run the GUI, as previously mentioned. Finally, within this directory, run the uvp.py script using the `uvp` alias described in the previous section. 

**Note:** Please make sure to turn on **Light Mode** on your local computer before running the script, as Dark mode may affect the visibility of certain GUI widgets.

Once the `uvp` alias is invoked, the frames for the RC visual (described below) will be generated, processed, and stored in a subdirectory called `frames`. This is a one-time process and may take a few minutes depending on number of geometries contained in the RC geometry file.  Once these initial steps are completed, the GUI will appear:

<img src="uvp_example_1.gif" alt="UVP GUI">

The components of the GUI, shown above, are the following:
* **2D RC Visual**: Depicts the RC geometry for each value of *s*, the reaction coordinate. The 2D RC image for each value of *s* is generated using modified source code from the Atomic Simulation Environment (ASE) Python library.<sup>3,4</sup>
* **RC Visual Control Panel**: Enables the user to control the value of *s*, thereby changing the 2D RC visual and the energy/curvature plots as well. The user can do this in five ways: moving the slider with their mouse, entering a new value of *s* in the text box (followed by pressing "Jump to s value" or just pressing <kbd>return</kbd> on the keyboard), animating the reaction in the forward ("Forward") or reverse ("Reverse") directions, moving to the next or previous *s* values ("Step Forward" or "Step Backward", respectively), or skipping to the key points along the reaction coordinate ("First Point", "Transition State", or "Last Point"). The frame rate for the animation can be controlled by specifying the rate in the text box on the bottom-left side of the window (25 fps by default). In addition, the user can also download the XYZ file for the RC geometry corresponding to the current value of *s* ("Download XYZ File") or examine the geometry in 3D with the ASE GUI ("View with ASE GUI"). For more information on how to use the ASE GUI, please refer to the following ASE documentation: https://ase-lib.org/ase/gui/gui.html.
* **RP Curvature and RC Energy Plots**: Interactive plots showing the RP curvature and RC energy as a function of *s*. When the red tracers are enabled, the user can more easily see which values of the curvature and energy correspond to a given value of *s*. To prevent the GUI from crashing, the tracers must be disabled prior to running the animation ("Disable Tracers" in the RC Visual Control Panel). Once disabled, the tracers can be re-enabled with "Enable Tracers". The interactive plots, which are generated using the wxmplot library, have many more features that are accesible to the user, and a comprehensive description of these features is provided in the following documentation: https://newville.github.io/wxmplot/overview.html.

The other example reactions in the `examples` subdirectory are the following:
* `example_2`: The C-C bond activation of acetonitrile, catalyzed by a [Ni(dmpe)](dmpe = Bis(dimethylphosphino)ethane) complex
* `example_3`: The hydrogen migration reaction of the FC(O)SH molecule
* `example_4`: Sharpless epoxidation reaction
