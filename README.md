# URVA Visualization Program (UVP)

## Program Overview
UVP is a user-friendly GUI designed for the visual mechanistic analysis of URVA output data, especially the curvature of the reaction path (RP) on the potential energy surface of the reaction complex (RC), the union of the reacting species. This interface enables users to interactively and simultaneously compare the change in the RP curvature with changes in the RC geometry and electronic energy as a function of the reaction coordinate. Cross-comparing the RP curvature with the RC properties, the user can better understand the chemical meaning of each URVA reaction phase, thereby enriching the mechanistic analysis of each elementary step of a given reaction mechanism. 

The GUI requires three input files: an RP curvature file (.DAT format), an RC energy file (.CSV format), and an RC geometry file (.XYZ format). These output files are generated by our group's URVA2025 program and its corresponding post-processing scripts, which are available upon request (contact email: ekraka@mail.smu.edu). Examples of these input files are provided in the examples directory and are also discussed below.

## Installation

Clone the repository to your local machine and access the repository
```Python
git clone git@github.com:SMU-CATCO/UVP.git
cd UVP
```
Create the uvp conda environment from the uvp.yml environment file
```Python
conda env create -f uvp.yml
```
Copy the uvp.py file to a directory of your choice, such as the home directory
```Python
cp uvp.py path/to/new/uvp/directory
```
In the ~/.bashrc file, include an alias to run the Python script:
```Python
alias uvp="python3 path/to/new/uvp/directory"
```
Afterwards, reload the ~/.bashrc file
```Python
source ~/.bashrc
```
